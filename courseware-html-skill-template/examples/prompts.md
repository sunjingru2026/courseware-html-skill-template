# Example Prompts

## Convert One Lesson

```text
Use the interactive-courseware-html skill to convert this lecture PPT into a single student-facing HTML page.
Preserve all important concepts, figures, examples, code, and outputs. Add at least two useful interactions,
a classroom quiz, AI-labeled explanations, extension exercises, and reference answers.
```

## Convert Multiple Lessons

```text
Use the interactive-courseware-html skill to convert Chapter 6's four PPT files into four HTML pages.
Keep the visual style consistent across all pages. Validate every visible code block and make every multi-line
code block copyable.
```

## Match an Existing Style

```text
Use the interactive-courseware-html skill. Match the colors and UI style of reference.html,
but do not copy its chapter structure. Convert the attached lesson slides into a polished interactive web lesson.
```

## Add Practice Links

```text
Use the interactive-courseware-html skill. Add this class-practice link after the quiz:
<CLASS_PRACTICE_URL>

Add this practice-platform link as a capsule-style button in the extension exercise heading:
<PRACTICE_PLATFORM_URL>
```

## Regenerate With Corrections

```text
Use the interactive-courseware-html skill to regenerate the HTML page.
Do not omit slide content. Preserve all important source figures and code-output examples.
Check every generated code snippet for syntax errors before finishing.
```
